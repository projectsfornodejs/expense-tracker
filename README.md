# expense-tracker
[text](https://roadmap.sh/projects/expense-tracker)

To start the expense tracker cli, follow the below steps:

# Install
npm i

# Run the CLI

# Add an item
node index.js add --description <description> --amount <amount>

# Delete an item
node index.js delete --id <id>

# List items
node index.js list

# Summary of expenses
node index.js summary

# Summary for a specific month
node index.js summary --month <numerical value of month>
